( 1 5 4 2 8 ) -> ( 1 4 5 2 8 ), Tukar sejak 5 > 4

( 1 4 5 2 8 ) -> ( 1 4 2 5 8 ), Tukar sejak 5 > 2

Pass Kedua:

( 1 4 2 5 8 ) -> ( 1 4 2 5 8 )

( 1 4 2 5 8 ) -> ( 1 2 4 5 8 ), Tukar sejak 4 > 2

( 1 2 4 5 8 ) -> ( 1 2 4 5 8 )

( 1 2 4 5 8 ) -> ( 1 2 4 5 8 )

Sekarang, array sudah diurutkan, tetapi algoritme kita tidak tahu apakah sudah selesai. Algoritma membutuhkan satu lintasan utuh tanpa pertukaran apa pun untuk mengetahui bahwa itu diurutkan.

Pass Ketiga:

( 1 2 4 5 8 ) -> ( 1 2 4 5 8 )

( 1 2 4 5 8 ) -> ( 1 2 4 5 8 )

( 1 2 4 5 8 ) -> ( 1 2 4 5 8 )

Java:

   public class BubbleSortExample {  

       static void bubbleSort(int[] arr) {  

           int n = arr.length;  

           int temp = 0;  

            for(int i=0; i < n; i++){  

                    for(int j=1; j < (n-i); j++){  

                             if(arr[j-1] > arr[j]){  

                                    //swap elements  

                                    temp = arr[j-1];  

                                    arr[j-1] = arr[j];  

                                    arr[j] = temp;  

                            }  

                             

                    }  

            }  

     

       }  

       public static void main(String[] args) {  

                   int arr[] ={4,2,5,3};  

                   

                   System.out.println("Array Before Bubble Sort");  

                   for(int i=0; i < arr.length; i++){  

                           System.out.print(arr[i] + " ");  

                   }  

                   System.out.println();  

                     

                   bubbleSort(arr);//sorting array elements using bubble sort  

                   

                   System.out.println("Array After Bubble Sort");  

                   for(int i=0; i < arr.length; i++){  

                           System.out.print(arr[i] + " ");  

                   }  

     

           }  

   }

func bubleSort(arr []int) {

 for {

   disortir := true

   for i := 0; i < len(arr)-1; i++ {

     if arr[i] > arr[i+1] {

       tmp := arr[i]

       arr[i] = arr[i+1]

       arr[i+1] = tmp

       disortir = false

     }

   }

   if disortir == true {

     break

   }

 }

}

Penjelasan:

Insertion sort

Terdapat deret bilangan: (1, 2, 5, 3, 4)

Tahap-tahap insertion sort terhadap deret ini:

Siklus 0

(1, 2, 5, 3, 4)

Siklus 1

Key/kunci ada pada elemen ke-2.

(1, 2, 5, 3, 4)

PERIKSA: Elemen ke-2 LEBIH BESAR DARI elemen ke-1, maka lanjutkan.

Siklus 2

Key/kunci ada pada elemen ke-3.

(1, 2, 5, 3, 4)

PERIKSA: Elemen ke-3 LEBIH BESAR DARI elemen ke-2, maka lanjutkan.

Siklus 3

Key/kunci ada pada elemen ke-4.

(1, 2, 5, 3, 4)

PERIKSA: Elemen ke-4 (3) TIDAK LEBIH BESAR DARI elemen ke-3 (5), maka elemen ke-3 ditukar (swap) dengan elemen ke-3.

(1, 2, 3, 5, 4)

PERIKSA: Elemen ke-3 (3) LEBIH BESAR DARI elemen ke-2 (2), maka lanjutkan.

Siklus 4

Key/kunci ada pada elemen ke-5.

(1, 2, 3, 5, 4)

PERIKSA: Elemen ke-5 (4) TIDAK LEBIH BESAR DARI elemen ke-4 (5), maka elemen ke-5 ditukar (swap) dengan elemen ke-4.

(1, 2, 3, 4, 5)

PERIKSA: Elemen ke-4 (4) LEBIH BESAR DARI elemen ke-3 (3), maka SELESAI.

22. Buatlah notasi algoritma menggunakan pseudocode untuk menghitung rata rata 20 bilangan yang berurutan￼


Untuk menghitung rata rata 20 bilangan yang berurutan, kita dapat menggunakan 2 metode yang diimplementasikan dalam notasi algoritma menggunakan pseudocode, seperti diuraikan di bawah ini.
_________________

Asumsi

Yang dimaksud dengan “20 bilangan berurutan“ pada pertanyaan ini adalah 20 bilangan bulat berurutan dengan selisih antar bilangan sama dengan 1.

Sebagai contoh: (1, 2, 3, ..., 20), atau (100, 101, 102, ..., 119).
_________________

NOTASI ALGORITMA dengan PseudocodeMetode 1: Dengan perulangan

[tex]\begin{aligned}&\underline{\texttt{PROGRAM HitungRataRata20Bilangan\_1}}\\&\underline{\texttt{Kamus}}\\&\quad\texttt{D{EF}INE jumlah, ratarata AS REAL}\\&\quad\texttt{D{EF}INE min, hitung AS INTEGER}\end{aligned}[/tex]
[tex]\begin{aligned}&\underline{\texttt{Algoritma}}\\&\quad\texttt{{IN}PUT min}\\&\quad\texttt{hitung $\leftarrow$ 0}\\&\quad\texttt{jumlah $\leftarrow$ 0}\\&\quad\texttt{WHILE hitung < 20 DO}\\&\quad\quad\texttt{jumlah $\leftarrow$ jumlah + min + hitung}\\&\quad\quad\texttt{hitung $\leftarrow$ hitung + 1}\\&\quad\texttt{END WHILE}\\&\quad\texttt{ratarata $\leftarrow$ jumlah/20}\\&\quad\texttt{{OUT}PUT ratarata}\\\end{aligned}[/tex]

Contoh penelusuran algoritma

Contoh 1 ⇒ Masukan: min = 1.

Iterasi WHILE ... END WHILE:
1 : jumlah = 0 + 1 + 0 = 1
2 : jumlah = 1 + 1 + 1 = 3
3 : jumlah = 3 + 1 + 2 = 6
4 : jumlah = 6 + 1 + 3 = 10
5 : jumlah = 10 + 1 + 4 = 15
6 : jumlah = 15 + 1 + 5 = 21
7 : jumlah = 21 + 1 + 6 = 28
8 : jumlah = 28 + 1 + 7 = 36
9 : jumlah = 36 + 1 + 8 = 45
10 : jumlah = 45 + 1 + 9 = 55
11 : jumlah = 55 + 1 + 10 = 66
12 : jumlah = 66 + 1 + 11 = 78
13 : jumlah = 78 + 1 + 12 = 91
14 : jumlah = 91 + 1 + 13 = 105
15 : jumlah = 105 + 1 + 14 = 120
16 : jumlah = 120 + 1 + 15 = 136
17 : jumlah = 136 + 1 + 16 = 153
18 : jumlah = 153 + 1 + 17 = 171
19 : jumlah = 171 + 1 + 18 = 190
20 : jumlah = 190 + 1 + 19 = 210Hitung ratarata: ratarata = 210 /20 = 10.5Output ratarata: 10.5

Contoh 2 ⇒ INPUT: min = 100.

Iterasi WHILE ... END WHILE:
1 : jumlah = 0 + 100 + 0 = 100
2 : jumlah = 100 + 100 + 1 = 201

37. Urutkan list bilangan 4 2 5 3 dengan algoritma bubble sort ​


Iterasi pertama:

(4 2 5 3) menjadi (2 4 5 3)

(2 4 5 3) menjadi (2 4 5 3)

(2 4 5 3) menjadi (2 4 3 5)

Iterasi kedua:

(2 4 3 5) menjadi (2 4 3 5)

(2 4 3 5) menjadi (2 3 4 5)

(2 3 4 5) menjadi (2 3 4 5)

Jadi, hasil dari algoritma bubble sort untuk 4 2 5 3 adalah 2 3 4 5.

Pembahasan:

Algortima bubble sort adalah teknik pengurutan sebuah data numerik yang tersusun secara salah, sehingga akan diurutkan dengan cara menukarkan angka pertama dan kedua, angka kedua dan ketiga, angka ketiga dan keempat, dan seterusnya.

Adapun proses pengurutan data dari bilangan 4 2 5 3 dengan algoritma bubble sort, yaitu:

Iterasi pertama:

(4 2 5 3) menjadi (2 4 5 3)

(4 > 2, maka 4 ditukar dengan 2).

(2 4 5 3) menjadi (2 4 5 3)

(Tidak ada penukaran karena 4 < 5).

(2 4 5 3) menjadi (2 4 3 5)

(5 > 3, maka 5 ditukar dengan 3).

Iterasi kedua:

(2 4 3 5) menjadi (2 4 3 5)

(Tidak ada penukaran karena 2 < 4).

(2 4 3 5) menjadi (2 3 4 5)

(4 > 3, maka 4 ditukar dengan 3).

(2 3 4 5) menjadi (2 3 4 5)

(Tidak ada penukaran karena 4 < 5).

Jadi, hasil dari algoritma bubble sort untuk 4 2 5 3 adalah 2 3 4 5.
