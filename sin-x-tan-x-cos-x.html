2. Quiz (+50): tan²(x)-sin²(x) = a. cos²(x)tan²(x) b. tan²(x)sin²(x) c. tan²(x)cos²(x) d. -cos²(x)sin²(x)


tan²(x) - sin²(x)

= sin²(x) / cos²(x) - sin²(x)

= sin²(x)/cos²(x) - sin²(x)cos²(x) / cos²(x)

= sin²(x) - sin²(x)cos²(x) / cos²(x)

= sin²(x) (1 - cos²(x)) / cos²(x)

= sin²(x) / cos²(x) × (1 - cos²(x))

= tan²(x) × sin²(x)

= tan²(x)sin²(x) (b.)
