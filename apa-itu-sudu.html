21. besar sudut pelurus dari sudu 154° adalah​


Jawaban:

26°

Penjelasan dengan langkah-langkah:

→ 180° - 154°

→ 26°

Catatan :

→ Jumlah sudut saling berpelurus = 180°

→ Jumlah sudut saling berpenyiku = 90°

Besar sudut pelurus dari sudut 154⁰ adalah

DiketahuiBesar Sudut ➡️ 154⁰

Ditanya

Besar Sudut Pelurus

Penyelesaian

x + 154⁰ = 180⁰

x = 180⁰ - 154⁰

x = 26⁰✔

Kesimpulan

Jadi, Besar sudut pelurus tersebut adalah 26⁰✔

26. perhatikan gambar besaran sudu x adalah​


Jawab:Nilai x adalah 38

Penjelasan dengan langkah-langkah:

agar lebih jelas, gambar ditambahkan huruf dan angka seperti gambar di paling bawah jawaban. (tidak perlu dibuat)

Dari gambar itu diketahui:

<A3 = 2x - 15

<B4 = 3x + 5

ditanya:

Nilai x?

dijawab:

nilai x tidak bisa langsung dicari karena <A3 dan <B4 tidak berhubungan langsung.

Untuk itu diperlukan perantara sudut yang berhubungan langsung dengan <A3 dan <B4.

Hubungan - hubungan sudut:

1. Sehadap.

<A1 = <B1

<A2 = <B2

<A3 = <B3

<A4 = <B4

2. Dalam bersebrangan.

<A3 = <B1

<A4 = <B2

3. Luar bersebrangan.

<A1 = <B3

<A2 = >B4

4. Bertolak belakang.

<A1 = <A3

<A2 = <A4

<B1 = <B3

<B2 = <B4

5. Dalam sepihak.

<A3 + <B2 = 180°

<A4 + <B1 = 180°

6. Luar sepihak.

<A1 + <B4 = 180°

<A2 + <B3 = 180°

Dari pernyataan di atas bisa ditemukan perantara antara <A3 dengan <B4 yaitu <A1 karena:

<A1 bertolak belakang dengan <A3

<A1 luar sepihak dengan <B4

Jadi:

<A1 = <A3

<A1 = 2x - 15

Nilai x adalah:

<A1 + <B4 = 180°

2x - 15 + 3x + 5 = 180°

2x + 3x = 180 + 15 - 5

5x = 190

x = [tex]\frac{190}{5}[/tex]

x = 38

Nilai x yang memenuhi adalah 38
