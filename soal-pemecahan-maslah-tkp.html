8. bagaimana cara memecahkan soal ini


Misal, akar-akarnya adalah x1 dan x2

Ingat bahwa :

x1 + x2 = -B/A = -a/1 = -a

x1.x2 = C/A = b/1 = b

Artinya :

(1 - √2) + x2 = -a

Karena a adalah bilangan bulat, maka haruslah x2 = (c + √2), untuk menghilangkan nilai √2.

Maka :

(1 - √2) + x2 = -a

(1 - √2) + (c + √2) = -a

c + 1 = -a

c = -a - 1

x1.x2 = b

(1 - √2)(c + √2) = b

Karena 1 < √2, maka (1 - √2) < 0

Karena (1 - √2) < 0, dan b merupakan bilangan real positif, maka (c + √2) < 0 (karena negatif kali negatif adalah positif)

Artinya, c + √2 < 0

c < -√2

Artinya :

c = -a - 1

-a - 1 < -√2

a + 1 > √2

a > √2 - 1

Jadi, nilai a terkecil adalah 1

26. Bagaimana cara memecahkan soal ini



[tex]3.87 \times 4.1 = \frac{387}{100} \times \frac{41}{10} = 15867[/tex]
SEMOGA BERMANFAAT YA KA.Saat mengkali, komanya hilangkan saja dulu.

387 x 41 = 15867

Lalu lihat angka dibelakang koma pada kedua bilangan tersebut.

Angka dibelakang koma pada bilangan 3,87 adalah 2 angka.

Angka dibelakang koma pada bilangan 4,1 adalah 1 angka.

Jadi, 2+1 = 3
Jawabannya harus 3 angka dibelakang koma

Maka pada jawabannya, yaitu 15867
Tambahkan 3 angka di belakang koma, jawabannya jadi 15,867
