Pembahasan :
Rumus turunan fungsi trigonometri, yaitu:
1. f(x) = sin x → f'(x) = cos x
2. f(x) = cos x → f'(x) = -sin x
3. f(x) = tan x → f'(x) = sec² x
4. f(x) = cotan x → f'(x) = -cosec² x
5. f(x) = sec x → f'(x) = sec x . tan x
6. f(x) = cosec x → f'(x) = -cosec x . cotan x

Mari kita lihat soal tersebut.
Turunan dari y = cos 2x adalah...

Jawab :
y = cos 2x
y' = 2 (-sin 2x)
⇔ y' = -2 sin 2x

Jadi, turunan dari y = cos 2x adalah y' = -2 sin 2x.

maka, dy/dc = -sin(c)

= (-sin(c))(-sin(b))(-sin(a))(2)

= -2sin(c).sin(b).sin(a)

= -2.sin(cos(b)).sin(b).sin(a)

35. Nyatakan sebagai jumlah sinus atau kosinus! 2 cos (2x + y) cos(2x - y)​


Bentuk penjumlahan dalam sinus atau kosinus pada perkalian trigonometri 2 cos (2x + y) cos (2x – y) adalah (cos 4x + cos 2y). Hasil tersebut diperoleh dengan menggunakan rumus:

2 cos A cos B = cos (A + B) + cos (A – B)

Penjelasan dengan langkah-langkah

Rumus perkalian trigonometri sinus dan kosinus.

2 sin A cos B = sin (A + B) + sin (A – B)2 cos A sin B = sin (A + B) – sin (A – B)2 cos A cos B = cos (A + B) + cos (A – B)–2 sin A sin B = cos (A + B) – cos (A – B)

Diketahui

2 cos (2x + y) cos (2x – y)

Ditanyakan

Nyatakan dalam bentuk penjumlahan sinus atau kosinus!

Jawab

Langkah 1

Misal:

A = 2x + yB = 2x – y

Langkah 2

A + B = (2x + y) + (2x – y)

         = 4x

A – B = (2x + y) – (2x – y)

         = 2x + y – 2x + y

         = 2y

Langkah 3

Jadi bentuk penjumlahan dari 2 cos (2x + y) cos (2x – y) adalah:

2 cos (2x + y) cos (2x – y)

= 2 cos A cos B

= cos (A + B) + cos (A – B)

= cos 4x + cos 2y
