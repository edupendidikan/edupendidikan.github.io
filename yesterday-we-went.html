Simple Past Tense

Jawaban:

(-) I did not go to school yesterday

PEMBAHASAN

Simple Past Tense adalah tenses yg digunakan utk menunjukkan Kegiatan yg terjadi di masa lampau.

=> Rumus Past Tense:

1. Nominal Sentence adalah kalimat yg tidak ada kata kerja di dalamnya

(+) S + to be (was/were) + O

(-) S + to be (was/were) + Not + O

(?) to be (was/were) + S + O?

=> Subjek + to be

• I, he, she, it + was

• you, we, they + were

2. Verbal Sentence adalah kalimat yg terdapat kata kerja (verb)

(+) S + Verb 2 + O

(-) S + did not/didn’t + Verb 1 + O

(?) Did + S + V1 + O?

=> Keterangan waktu Past Tense

• yesterday

• last night

• two days ago

Kesimpulan:

(+) pada kalimat Positive menggunakan V2 (went)

(-) (?) tapi pada kalimat Negative dan Interrogative menggunakan V1 (go) karena sudah ada Auxiliary DID.

Jadi yg benar Berdasarkan Rumus

(-) S + did not/didn’t + V1 + O

= I did not go to school yesterday

PEMBAHASANPast Tense

Digunakan untuk menyatakan Kegiatan yang terjadi di masa lalu atau Peristiwa masa lampau.

Ciri Past Tense=> yesterday ✔️=> Last night=> a few minutes ago=> dll.

Rumus Past Tense

(+) S + Verb-2

(-) S + didn’t + Verb-1

(?) Did + S + Verb-1?

Keterangan:A. go => Verb-1B. goes => Verb+esC. went => Verb-2D. gone => Verb-3

KESIMPULAN

➡️ Keterangan waktu lampau “yesterday” (kemarin)

➡️ Maka menggunakan Verb-2 “went” (c) ✔️

    Subject + verb 2

Untuk bentuk negasinya ditambahkan did not dan menggunakan verb 1

    Subject + did not + verb 1

Untuk bentuk interogatifnya ditambahkan did di depan dan menggunakan verb 1

    Did + Subject + verb 1 + ?
