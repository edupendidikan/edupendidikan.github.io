[tex]D_{f\cdot g}=\{x|x\ne -\frac{1}{3} ,2\}[/tex]

Penjelasan:

Diketahui fungsi

[tex]f(x)=\frac{2x-1}{x-2} \\\\g(x)=\frac{x}{3x+1}[/tex]

[tex](f\cdot g)(x)=f(x)\cdot g(x)\\\\(f\cdot g)(x)= \frac{2x-1}{x-2} \cdot \frac{x}{3x+1}\\\\(f\cdot g)(x)=\frac{(2x-1)x}{(x-2)(3x+1)}[/tex]

Penyebut dari fungsi rasional (pecahan) tersebut adalah (x - 2)(3x + 1).

Agar pecahan mempunyai nilai bilangan real, maka penyebut tidak boleh sama dengan 0.

(x - 2)(3x + 1) ≠ 0

x - 2 ≠ 0 atau 3x + 1 ≠ 0 maka

x ≠ -2 atau x ≠ -1/3

Jadi, daerah asal/domain dari fungsi (f × g)(x) adalah:

[tex]D_{f\cdot g}=\{x|x\ne -\frac{1}{3} ,2\}[/tex]

Penjelasan dengan langkah-langkah:

x = 2 --> f(2) = 2 + 2 = 4

x = 3 --> f(3) = 3 + 2 = 5

x = 4 --> f(2) = 4 + 2 = 6

x = t --> f(t) = t + 2 = t + 2

Jika x diganti g(x), diperoleh

f (g(x)) = x³ + 2 = x³ + 2

Misalkan fungsi h(x) = f ( g(x)) = x³ + 2

Dengan cara yang sama, maka

g (f(x)) = (x + 2)³

Misalkan fungsi f dirumuskan dengan f(x) = x + 2 dan g dirumuskan dengan g(x) = x³.

Dengan menggunakan rumus f(x) = x + 2, untuk

x = 1 >>>>> f(1) = 1 + 2 = 3x = 2 >>>>> f(2) = 2 + 2 = 4x = 3 >>>>> f(3) = 3 + 2 = 5x = t >>>>> f(t) = t + 2

Jika x diganti dengan g(x), diperoleh

f(g(x)) = g(x) + 2 = x³ + 2

Misalkan h(x) = f(g(x)) = f(x³) = x³ + 2

Fungsi h(x) yang diperoleh dengan cara diatas, dinamakan fungsi komposisi g dan f. Fungsi ini ditulis dengan f o g dibaca "f bundaran g" .

Dengan cara yang sama, maka

g(f(x)) = g(x + 2) = (x + 2)³

Fungsi g(f(x)) kemudian ditulis (g o f)(x).

Berdasarkan hal tersebut, disumpulkan bahwa :

(g o f)(x) = g(f(x))
