8. Arsen dalam 1,010 g sampel pestisida dilarutkan dalam asam menjadi H3AsO4. Setelah asam arsenat dinetralkan lalu ditambahkan 40,00 mL larutan AgNO3 0,00622 M membentuk endapan Ag3AsO4. Ion Ag+ yang tersisa dititrasi dengan 10,76 mL larutan KSCN. Berapakah persentase (%b/b) As2O3 dalam sampel?​


Untuk menentukan persentase (%b/b) As2O3 dalam sampel, kita perlu mengikuti langkah-langkah berikut:

Langkah 1: Menghitung jumlah mol AgNO3 yang bereaksi dengan arsen dalam larutan.

Konsentrasi AgNO3 = 0,00622 M
Volume AgNO3 = 40,00 mL = 0,04000 L

Jumlah mol AgNO3 = konsentrasi x volume = 0,00622 M x 0,04000 L = 0,0002488 mol AgNO3

Karena AgNO3 bereaksi dengan arsen dalam larutan dengan perbandingan 1:1, jumlah mol arsen (As) dalam larutan juga 0,0002488 mol.

Langkah 2: Menghitung jumlah mol As2O3 dalam sampel.

Dalam senyawa Ag3AsO4, perbandingan Ag:AsO4 adalah 3:1. Oleh karena itu, jumlah mol arsen (As) dalam sampel adalah 1/3 dari jumlah mol AgNO3.

Jumlah mol As2O3 = 1/3 x 0,0002488 mol = 0,00008293 mol As2O3

Langkah 3: Menghitung massa As2O3 dalam sampel.

Massa molar As2O3 = (2 x Ar As) + (3 x Ar O) = (2 x 74,92 g/mol) + (3 x 16,00 g/mol) = 197,84 g/mol

Massa As2O3 = jumlah mol x massa molar = 0,00008293 mol x 197,84 g/mol = 0,0164 g As2O3

Langkah 4: Menghitung persentase (%b/b) As2O3 dalam sampel.

Persentase (%b/b) As2O3 = (massa As2O3 / massa sampel) x 100%
Massa sampel = 1,010 g (diberikan dalam pertanyaan)

Persentase (%b/b) As2O3 = (0,0164 g / 1,010 g) x 100% = 1,62%

Jadi, persentase (%b/b) As2O3 dalam sampel adalah 1,62%.

28. pada masing masing perubahan berikut tentukan peristiwa oksidasi dan reduksi dan berikan alasannya. a. CrO2- menjadi CrO42-b. BaSO4 menjadi BaS c. H2S menjadi S d. OcI-menjadi CLO3-e. AsH menjadi H3AsO4​


CrO₂⁻² menjadi CrO₄⁻² adalah peristiwa oksidasi. ClO⁻ menjadi ClO₃⁻ adalah peristiwa oksidasi.BaSO₄ menjadi BaS adalah peristiwa reduksi. AsH₃ menjadi H₃AsO₄ adalah peristiwa oksidasi. H₂S menjadi S adalah peristiwa oksidasi.

Pembahasan

∴CrO₂⁻² menjadi CrO₄⁻² adalah peristiwa oksidasi.

CrO₂⁻²

Biloks Cr + 2 Biloks O = -2

Biloks Cr + 2 x (-2) = -2

Biloks Cr -4 = -2

Biloks Cr = +2

CrO₄⁻²

Biloks Cr + 4 Biloks O = -2

Biloks Cr + 4 x (-2) = -2

Biloks Cr - 8 = -2

Biloks Cr = +6

⇒ Alasanya adalah Biloks Cr naik dari +2 menjadi +6

∴ClO⁻ menjadi ClO₃⁻ adalah peristiwa oksidasi.

ClO⁻

Biloks Cl + Biloks O = -1

Biloks Cl + 1 x (-2) = -1

Biloks Cl - 2 = -1

Biloks Cr = +1

ClO₃⁻

Biloks Cr + 3 Biloks O = -1

Biloks Cr + 3 x (-2) = -1

Biloks Cr - 6 = -1

Biloks Cr = +5

⇒ Alasanya adalah Biloks Cl naik dari +1 menjadi +5

∴ BaSO₄ menjadi BaS adalah peristiwa reduksi.

BaSO₄

Biloks S + Biloks Ba + 4 Biloks O = 0

Biloks S + 2 + 4 x (-2) = 0

Biloks S + 2 -8 = 0

Biloks S = +6

BaS

Biloks S + Biloks Ba = 0

Biloks S + 2 = 0

Biloks S = -2

⇒ Alasanya adalah Biloks S turun dari +6 menjadi -2

∴AsH₃ menjadi H₃AsO₄  adalah peristiwa oksidasi.

AsH₃

Biloks As + 3 Biloks H = 0

Biloks As + 3 x 1= 0

Biloks As + 3 = 0

Biloks As = -3

H₃AsO₄

Biloks As + 3 Biloks H + 4 Biloks O= 0

Biloks As + 3 x 1+ 4 x (-2)= 0

Biloks As + 3 -8 = 0

Biloks As = +5

⇒ Alasanya adalah Biloks S naik dari -3 menjadi +5

∴H₂S menjadi S adalah peristiwa oksidasi.

H₂S

Biloks S + 2 Biloks H = 0

Biloks S + 2 x 1= 0

Biloks S + 2 = 0
