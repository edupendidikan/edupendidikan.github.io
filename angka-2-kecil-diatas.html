15. Banyak bilangan yang terdiri atas 2 angka yang dapat dibentuk dari angka 3,6 dan 8?


Kombinasi

Untuk mengetahui banyaknya bilangan yang dapat dibentuk dari angka 3, 6, dan 8 dan dapat dibentuk 2 angka adalah dengan menggunakan rumus kombinasi, karena tidak ada aturan lain selain banyaknya 2 bilangan yang dapat dibentuk.

C(n,r) = n! / ((n-r)!r!)

= 3! / ((3-2)!2!)

= 3! / (1! x 2!)

= 3

Maka, banyaknya bilangan yang dapat dibentuk adalah 3

dicari dengan rumus permutasi
2 dari 3 angka
= 3 P 2
= ( 3 ! )/( 3−2)!
= 3×2×1
= 6

26. Sebuah bilangan terdiri atas 2 angka. Bilangan itu adalah 3 kurangnya dari empat kali jumlah angka-angkanya. Dan angka kedua dikurangi angka pertama sama dengan 2. Tentukan bilangan tersebut


Berikan permisalan bahwa bilangan tersebut adalah ab
Dengan basis 10, dapat diberikan dengan:
10a + b, untuk 0 < a ≤ 9, dan 0 ≤ b ≤ 9

Bilangan itu adalah 3 kurangnya dari 4 kali jumlah angkanya (dalam hal ini jumlah angka diberikan dalam a+b)
Maka itu:
10a + b = 4(a+b) - 3
10a + b = 4a + 4b - 3
6a - 3b = -3
2a - b = -1
Lakukan substitusi dengan:
b = 2a + 1 (i)

Sera, angka kedua (satuan) dikurangi dengan angka pertama (puluhan) adalah 2.
b - a = 2

Untuk itu, substitusi (i) pada persamaan ini
2 = b - a
2 = (2a + 1) - a
2 = a + 1
a = 2 - 1
a = 1

Serta, didapat pula:
b = 2a + 1
b = 2(1) + 1
b = 3

Maka, bilangan tersebut adalah 10a + b = 10(1) + 3 = 13
