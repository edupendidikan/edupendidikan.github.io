24. berikan contoh text procedure how to operate computer


Contoh teks procedure "how to operate computer":

Operating a computer:

First, make sure that the computer CPU is connected to the power source which here in Indonesia is 220VAC. After the computer CPU is connected to the power source, switch the power button on. You can find the power button on the front side of the CPU and wait until the computer finishes the booting process. Next, the computer will show the start menu for Windows operating system and will ask you to input the password for your Windows account.

Pembahasan

Procedure text di atas bisa kita terjemahkan sebagai berikut.

Pastikan terlebih dahulu bahwa CPU komputer terhubung ke sumber listrik yang di Indonesia adalah 220VAC. Setelah CPU komputer tersambung ke sumber listrik, tekan tombol on. Anda bisa menemukan tombol on di sisi depan CPU dan tunggu hingga komputer menyelesaikan proses booting. Selanjutnya, komputer akan menampilkan menu start untuk sistem operasi Windows dan akan meminta Anda memasukkan kata sandi untuk akun Windows Anda.
