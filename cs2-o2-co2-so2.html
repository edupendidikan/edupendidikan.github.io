6. Reaksi pembakaran cs2 menghasilkan co2 dan so2. jika dilakukan pembakaran sempurna terhadap 3 l cs2, maka akan dihasilkan so2 sebanyak


Entalpi pembakaran 0,1 mol SO₂ adalah -87,18 kJ. Perubahan entalpi Standar Pembakaran adalah perubahan entalpi dari reaksi pembakaran 1 molar senyawa atau unsur. Pada reaksi ini, senyawa harus terbakar sempurna, yang ditandai dengan pengikatan oksigen.

Penjelasan dengan langkah-langkah:

Diketahui:

ΔHf CO₂ = -393,5

ΔHf SO₂ = 296,8

ΔHf CS₂ = 115,3

ΔHf O₂ = 0

Ditanya: Entalpi pembakaran CS₂?

Jawab:

ΔH = (ΔHf CO₂ + 2 ΔHf SO₂) - (ΔHf CS₂ + 3ΔHf O₂)

ΔH = (-393,5 + 2 (296,8)) - (115,3 + 0)

ΔH  = -987, 1 + 115,3

ΔH  = -871,8 kJ/mol

Mol CS₂ = 7,6 : 76 = 0,1 mol

Entalpi pembakaran untuk 0,1 mol CS₂ yaitu: 0,1 mol x (-871,8 kJ/mol) = 87,18 kJ

Jadi, besar entalpi pembakaran untuk 0,1 mol CS₂ yaitu: 0,1 mol x (-871,8 kJ/mol) = 87,18 kJ. Perubahan entalpi Standar Pembakaran adalah perubahan entalpi dari reaksi pembakaran 1 molar senyawa atau unsur. Pada reaksi ini, senyawa harus terbakar sempurna, yang ditandai dengan pengikatan oksigen.

18. Jika diketahui:C + 2S → CS2 delta H = +27,55 kkalC + O2 → CO2 delta H = -94 kkalS + O2 →SO2 delta H = -70,9 kkalBerapa kalor yang dilepaskan jika 9 gram CS2 dibakar sempurna? (Ar C = 12, S = 32)


Jika diketahui:

C + 2S ⇒ CS₂            ΔH = +27,55 kkal

C + O₂ ⇒ CO₂           ΔH = -94 kkal

S + O₂ ⇒ SO₂            ΔH = -70,9 kkal

Maka kalor yang dilepaskan jika 9 gram CS₂ dibakar sempurna (Ar C = 12g/mol, S = 32 g/mol) adalah 31,18 kkal/ mol.

Penyelesaian Soal :

Diketahui : C + 2S ⇒ CS₂            ΔH = +27,55 kkal

                  C + O₂ ⇒ CO₂           ΔH = -94 kkal

                  S + O₂ ⇒ SO₂            ΔH = -70,9 kkal

                  massa CS₂ = 9 gram

                  Ar C = 12g/mol, S = 32 g/mol

Ditanya : Kalor (Q) ?

Jawab :

Soal tersebut mengenai hukum Hess dengan jenis reaksi bertingkat. maka dari reaksi tersebut untuk didapat hasil reaksi target, yaitu pembakaran CS₂

Reaksi akhir ⇒ CS₂ + 3O₂ → CO₂ + 2SO₂     (sudah setara)

Langkah 1 : Hitung nilai ΔH reaksi dengan menggunakan rumus hukum Hess :

C + 2S ⇒ CS₂            ΔH = +27,55 kkal     ║ dibalik

C + O₂ ⇒ CO₂           ΔH = -94 kkal

S + O₂ ⇒ SO₂            ΔH = -70,9 kkal          ║ ×2

CS₂ ⇒  C + 2S                   ΔH = -27,55 kkal  

C + O₂ ⇒ CO₂                    ΔH = -94 kkal

___________________________________ +

CS₂ + O₂ ⇒ CO₂  + 2S            ΔH = -121,55 kkal  

2S + 2O₂ ⇒ 2SO₂                   ΔH = -141,8 kkal

___________________________________ +

CS₂ + 3O₂ ⇒  CO₂  +  2SO₂   ΔH = -263,35 kkal

Dalam mengeliminasi reaksi maka harus diperhatikan reaksi akhirnya. Letak setiap zat direaksi akhir akan berpengaruh.

Reaksi 1, diubah posisi CS₂ yang mulanya dihasil reaksi menjadi dipereaksi. maka tanda ΔH juga harus diubah.Sedangkan reaksi 2, tidak diubah dan tetap.Sedangkan pada reaksi 3, dikalikan dengan dua, hal ini dikarenakan sesuai dengan reaksi target menginginkan 2 mol SO₂.

Langkah 2 : Hitung massa molekul relatif (Mr) CS₂ dengan menggunakan cara sebagai berikut :

Mr CS₂ = Ar C + (2 × Ar S)

            = 12 g/mol + (2 × 32 g/mol)

            = 12 g/mol + 64 g/mol

            = 76 g/mol

Langkah 3 : Hitung mol CS₂ dengan menggunakan cara sebagai berikut :

mol CS₂ = massa/ Mr

              = 9 gram / 76 g/mol

              = 0,118 mol

Langkah 4 : Hitung Kalor (Q) yang dilepaskan dari reaksi pembakaran dengan cara sebagai berikut :

Q = mol × ΔH

   = 0,118 mol × (-263,35 kkal )

   = -31,18 kkal/ mol

Tanda minus pada ΔH menunjukkan jenis reaksi eksoterm (kalor dilepaskan). Hal ini juga sesuai dengan pertanyaan, yang menanyakan jumlah kalor yang dilepaskan.

∴ Kesimpulan kalor yang dilepaskan dari reaksi pembakaran CS₂ adalah 31,18 kkal/ mol.

Pembahasan :

Hukum Hess

Tidak semua reaksi kimia berlangsung dalam satu tahap, contohnya reaksi pembuatan belerang (baik melalui proses kontak maupun kamar timbal) dan reaksi pembuatan besi dari biji besi. Namun, menurut Hess (1840) berapa pun tahap reaksinya, jika bahan awal dan hasil akhirnya sama, akan memberikan perubahan entalpi yang sama.

CO₂ → C + O₂ ..........................ΔH = 394Kj
SO₂ → S + O₂ ..........................ΔH = 297 Kj

Reaksi pembentuka CS₂ adalah :
C + 2S → CS₂

Untuk mendapatkan reaksi pembentukan CS₂ maka:
Reaksi (1) dibalik, sehingga ΔH berubah tanda
Reaksi (2) dibalik, sehingga ΔH berubah tanda
Realsi (3) dibalik dan dikali, sehingga ΔH berubah tanda dan dikali 2

Maka reaksinya menjadi :
CO₂ + 2SO₂ → CS₂+ 3O₂ ......ΔH = 1110Kj
C + O₂ → CO₂ ..........................ΔH = -394Kj
2S + 2O₂ → 2SO₂ ....................ΔH = -594 Kj

Jumlahkan semua reaksi dan ΔH nya, coret senyawa yang sama antara sebelah kanan dan kiri tanda panah. Maka :
CO₂ + 2SO₂ → CS₂+ 3O₂ ......ΔH = 1110Kj
C + O₂ → CO₂ ..........................ΔH = -394Kj
2S + 2O₂ → 2SO₂ ....................ΔH = -594 Kj
________________________________ +
C + 2S → CS₂ ............................ΔH = 122 Kj


Sehingga, kalor pembentukan CS₂ adalah 122 Kj

28. Hitunglah entalpi pembakaran CS2 pada reaksi : CS2(g)+O2(g)--->CO2(g)+SO2(g) Jika diketahui ΔHf° CS2(9) = 117,1 kJ/mol ΔHf° CO2(g) = -393,5 kJ/mol ΔHf° SO2(g) = -296,8 kJ/mol


Jawaban:

Hitunglah entalpi pembakaran CS₂ pada reaksi :

CS₂ (g) + O₂ (g) ⇒ CO₂ (g) + SO₂ (g)

Jika diketahui

ΔHf° CS₂ (g) = 117,1 kJ/mol

ΔHf° CO₂ (g) = -393,5 kJ/mol

ΔHf° SO₂ (g) = -296,8 kJ/mol

hasil entalpi pembakaran CS₂ adalah -1104,2 kJ/mol.

Penyelesaian Soal :

Diketahui : ΔHf° CS₂ (g) = 117,1 kJ/mol

                  ΔHf° CO₂ (g) = -393,5 kJ/mol

                  ΔHf° SO₂ (g) = -296,8 kJ/mol

Ditanya : entalpi pembakaran CS₂ ?

Jawab :

LANGKAH PERTAMA (I)

Setarakan reaksi berikut :

CS₂ (g) + 3O₂ (g) ⇒ CO₂ (g) + 2SO₂ (g)

LANGKAH KEDUA (II)

Hitung ΔHf hasil reaksi dengan cara sebagai berikut :

ΔHf hasil reaksi = ΔHf° CO₂ + (2 × ΔHf° SO₂)

                          = (-393,5 kJ/mol) + (2 × (-296,8 kJ/mol))

                          = (-393,5 kJ/mol) + (-593,6 kJ/mol)

                          = -987,1 kJ/mol

LANGKAH KETIGA (III)

Hitung ΔHf pereaksi dengan cara sebagai berikut :

ΔHf pereaksi = ΔHf° CS₂ + (5 × ΔHf° O₂)

                      = 117,1 kJ/mol + (5 × 0)

                      = 117,1 kJ/mol

LANGKAH KEEMPAT (IV)

ΔH = Σ ΔHf hasil reaksi – Σ ΔHf pereaksi

     = (-987,1 kJ/mol) - 117,1 kJ/mol

     = -1104,2 kJ/mol

∴ Kesimpulan entalpi pembakaran CS₂ adalah -1104,2 kJ/mol.

Pembahasan :

Berdasarkan Entalpi Pembentukan Standar

Data dari entalpi pembentukan standar dapat juga digunakan untuk menghitung H reaksi (ΔH). Zat zat pereaksi mengurai membentuk unsur unsurnya, kemudian unsur unsur hasil uraian tersebut membentuk zat baru. Rumus yang digunakan adalah :

ΔH = Σ ΔHf hasil reaksi – Σ ΔHf pereaksi
