#SelamatBelajar Present Perfect TenseDoni has read the newspaper already

PembahasanPresent Perfect Tense

➡️ Digunakan untuk menyatakan kegiatan yang sudah selesai tapi masih ada kaitannya dengan masa sekarang.

Ciri Present Perfect

=> Since

=> For

=> Already ✔️

=> dll.

Rumus Present Perfect

(+) S + has/have + V3 + O

(-) S + has/have + not + V3 + O

(?) has/have + S + V3 + O?

CatatanI/you/we/they + have ✔️He/she/it/Doni + has

KESIMPULAN

Pilihan ganda yang sesuai:

a. Will read ➡️ will + V1 b. Reads ➡️ Verb+s c. Read ➡️ Verb-1 d. Has read ➡️ Has + V3 ✔️ e. is reading ➡️ be + V-ing

Keterangan:

Mungkin Maksudnya yg D adalah Has+read ✔️

24. Bentuk kata kerja Verb ke 3 dan Verb ke 2 dari Read , study , start, receive , already,never, (Tolong bantu secepatnya )


Verb 1 disebut juga dengan present verb, verb 2 disebut juga dengan past verb, sedangkan verb 3 disebut juga dengan past participle verb.

Verb 2 digunakan untuk past tense, sedangkan verb 3 digunakan untuk perfect tense dan passive voice.

Pembahasan

- read (present verb) - read (past verb) - read (past participle)

- study (present verb) - studied (past verb) - studied (past participle)

- start (present verb) - started (past verb) - started (past participle)

- receive (present verb) - received (past verb) - received (past participle)

- already (termasuk dalam adverb, sehingga tidak bisa diubah menjadi past verb dan past participle)

- never (termasuk dalam adverb, sehingga tidak bisa diubah menjadi past verb dan past participle)
