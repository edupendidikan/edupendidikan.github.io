20. dalam sebuah tes matematika yang berjumlah 40 soal mempunyai aturan sebagai berikut benar skor 4 salah skor -2 tsk di jawab -1 dian mengerjakan soal dengan salah 9 soal dan memperoleh slor 81 maka jawaban yang benar adalah​


Diket

Skor = 4b -2s - t,

b + s + t = 40

s = 9 Skor = 81

Tanya: b = ?

Jawab:

(1) Skor = 4b - 2s - t = 4b - 18 - t = 81

4b - t = 81 + 18

4b - t = 99

(2) b + s + t = 40

b + 9 + t = 40

b + t = 31 → t = 31 - b

(1)(2)

4b - (31 - b) = 99

5b = 99 + 31

b = 130 : 5

b = 26

Jadi jumlah jawaban benar Dian = 26.

23. 20point...Soalnya simple...|x - 6| = |x| + 6tentukan HPsaya menemukan HP{0}Tapi setelah saya tes, HP {x <= 0}ada yang bisa jelasin?


|x-6| = |x|+6
|x| = |x-6|-6
|x| ≥ 0
|x-6|-6≥0
x²-12x-36-36 ≥0
x²-12x ≥0
x(x-12)≥0
x≤0 atau x≥12
Tinggal dicek batasannya.Ikutan..

|x - 6| = |x| + 6
|x - 6| - |x| = 6

•
|x - 6|
x - 6 ---> x - 6 ≥ 0 ---> x ≥ 6
-(x - 6) ---> x - 6 < 0 ---> x < 6

|x|
x --> x ≥ 0
-x ---> x < 0

Kondisi 1 : x < 0
|x - 6| - |x| = 6
-(x - 6) - (-x) = 6
6 = 6
memenuhi utk x < 0

Kondisi 2 : 0 ≤ x < 6
-(x - 6) - x = 6
-2x = 0
x = 0

Kondisi 3 : x ≥ 6
x - 6 - x = 6
-6 ≠ 6 tdk memenuhi

Nilai x yg memenuhi :
x = 0 atau x < 0

Tes masuk SMA Negeri 3 terdiri dari 100 soal jawaban yang benar diberi skor 4, jawaban salah diberi skor -1, dan soal yang tidak dijawab diberi skor 0. dalam tes tersebut seorang calon siswa dapat menjawab soal dengan benar 75 soal, dan 5 soal tidak terjawab. Berapakah skor yang diperoleh calon siswa tersebut?​


Jawab:

280

Penjelasan dengan langkah-langkah:

b = 4

s = - 1

x = 0

Total soal 100

75 soal benar, 5 tidak dijawab. Sisanya salah sebanyak 100 -75-5 = 20 soal

Skor siswa = 75b + 20s + 5x

= 75.4 + 20.(-1) + 5.0

= 300 - 20 + 0

= 280
