4. Mengapa ip memakai /24


Jadi, jika sebuah network menggunakan prefix-length /24; maka berarti network tersebut menggunakan 24 bit pertama IP address sebagai network-portion, dan sisa 8 bit IP address terakhirnya merupakan host-portion.

Contoh, sebuah network dengan prefix-length 24; 192.168.1.0/24, maka :

Jumlah bit network-portion = 24 bit.Jumlah bit host-portion = 32 – 24 = 8 bit.

5. Apa itu IP? Apa itu IP Address? Apa itu IP Addressing? (Ini sama atau beda sih) Fungsinya itu semua buat apa? Lah kok ada /24, /23, ada 255.255.255.0, ini maksudnya apa? IP Address si fulan sama anton kok bisa sama? Gapapa? Ini katanya ip address saya 192.168.100.10, kok dilihat di internet malah beda?


Penjelasan:

Alamat IP (Internet Protocol Address atau sering disingkat IP) adalah deretan angka biner antara 32 bit sampai 128 bit yang dipakai sebagai alamat identifikasi untuk tiap komputer host dalam jaringan Internet. Panjang dari angka ini adalah 32 bit (untuk IPv4 atau IP versi 4), dan 128 bit (untuk IPv6 atau IP versi 6) yang menunjukkan alamat dari komputer tersebut pada jaringan Internet berbasis TCP/IP.

Sistem pengalamatan IP ini terbagi menjadi dua, yakni:

IP versi 4 (IPv4)

IP versi 6 (IPv6)

Pengiriman data dalam jaringan TCP/IP berdasarkan IP address komputer pengirim dan komputer penerima. IP address memiliki dua bagian, yaitu alamat jaringan (network address) dan alamat komputer lokal (host address) dalam sebuah jaringan.

Alamat jaringan digunakan oleh router untuk mencari jaringan tempat sebuah komputer lokal berada, sementara alamat komputer lokal digunakan untuk mengenali sebuah komputer pada jaringan lokal.

Informasi ini bisa diketahui dengan mengkombinasikan IP address dengan 32 bit angka subnet mask. IP address memiliki beberapa kelas berdasarkan kapasitasnya, yaitu Class A dengan kapasitas lebih dari 16 juta komputer, Class B dengan kapasitas lebih dari 65 ribu komputer, dan Class C dengan kapasitas 254 komputer.

18. Berikut ini pernyataan yang benar, kecuali ... IP address 192.168.0.1/24 termasuk kelas C Opsi IP address 192.168.0.1/24 memiliki default subnetmask 255.255.255.0 IP address 192.168.0.1/24 memiliki alamat network 192.168.0.0 IP address 192.168.0.1/24 memiliki alamat broadcast 192.168.0.255​


Jawaban:

Pernyataan yang benar adalah: "IP address 192.168.0.1/24 termasuk kelas C".

Pernyataan yang salah adalah: "IP address 192.168.0.1/24 memiliki default subnetmask 255.255.255.0". Sebab, 255.255.255.0 adalah subnet mask default untuk kelas C, namun itu bukan berarti setiap alamat IP kelas C harus menggunakan subnet mask tersebut. Dalam hal ini, subnet mask 192.168.0.1/24 adalah 255.255.255.0, tetapi bisa saja menggunakan subnet mask yang berbeda.

Sementara itu, pernyataan "IP address 192.168.0.1/24 memiliki alamat network 192.168.0.0" dan "IP address 192.168.0.1/24 memiliki alamat broadcast 192.168.0.255" keduanya benar. Hal ini karena alamat jaringan (network address) selalu merupakan alamat IP pertama dalam subnet, sementara alamat broadcast selalu merupakan alamat IP terakhir dalam subnet. Dalam kasus ini, alamat jaringan adalah 192.168.0.0, sedangkan alamat broadcast adalah 192.168.0.255.

22. Mengapa pada ip terdapat ( /24 ) fungsi nya apa?


Sebagai contoh jika kita memiliki IP Address 192.168.4.1 /24 (prefix 24) maka menurut pembahasan sebelumnya 24 bit pertama adalah porsi Network-ID dan 8 bit setelahnya adalah porsi Host-ID. Cara mendapatkan subnet mask nya cukup mudah yaitu

Jika bit Ip address tersebut adalah porsi Network-id maka semua nilai bit nya adalah SATU.

Jika bit Ip address tersebut adalah porsi Host-id maka semua nilai bit nya adalah NOL.

Maka Subnet Mask dari IP Address ber Prefix /24 adalah 255.255.255.0

Fungsinya adalah untuk menghitung Subnet Mask

26. Network id, broadcast id, range ip, jumlah host dari ip 192.168.45.1/24


IP address : 192.168.45.1/24 atau

= 192.168.45.1 255.255.255.0

Cara menghitung :

Ubah IP Address dan Netmask kedalam bentuk biner, 192.168.45.1 IP Address tersebut berada di kelas C,  ambil octet ketiga dan ke empat untuk dihitung, maka :

IP address : x.x.0010 1101. 0000 0001 = 45.1

Netmask : x.x.1111 1111. 0000 0000 = 255.0

Jawab:

1. Network ID

Network ID = IP Address AND Netmask

IP address : x.x.0010 1101. 0000 0001

Netmask    : x.x.1111 1111. 0000 0000

——————————————————

AND           : x.x.0010 1101. 0000 0000 = 45.0

Sehingga Network ID nya adalah 192.168.45.0

2. Broadcast ID

Broadcast ID = IP Address OR ~Netmask

IP address : x.x.0010 1101. 0000 0001

~Netmask  : x.x.0000 0000 . 1111 1111

———————————————————

OR : x.x.0010 1101. 1111 1111 = 45.255

Sehingga Broadcast Id nya adalah 192.168.45.255

3. Range IP

Range IP = Network ID hingga(-) Broadcast ID

Sehingga Range IP nya adalah 192.168.45.0 – 192.168.45.255

4. Jumlah Host

Jumlah Host = [tex]2^{y} -2[/tex]

dimana y adalah jumlah biner 0 pada octet 3 dan 4 netmask

/24 bila di-binerkan menjadi:

1111 1111. 1111 1111. 1111 1111. 0000 0000

255         255        255            0

Sehingga Jumlah Host

= [tex]2^{8} - 2[/tex]

= [tex]256 - 2[/tex]

= [tex]254[/tex]

28. Tentukan network ID pada IP 192.168.20.1/24


Jawaban:

Dalam notasi IP 192.168.20.1/24, angka 24 pada akhir blok menunjukkan jumlah bit 1 yang ada pada network portion, atau dapat dikatakan jumlah bit yang digunakan untuk mengidentifikasi network ID. Dalam hal ini, karena menggunakan 24 bit untuk network ID maka tinggal 8 bit lagi untuk host ID.

Untuk menentukan network ID dari IP tersebut, perlu dilakukan operasi AND antara setiap oktet IP dengan subnet mask. Subnet mask 24 bit adalah 255.255.255.0, yang dapat ditulis dalam bentuk binner sebagai 11111111.11111111.11111111.00000000.

Maka jika dilakukan operasi AND dengan IP 192.168.20.1:

```

192.168.20.1 -> 11000000.10101000.00010100.00000001 (dalam biner)

& subnet mask (24) -> 11111111.11111111.11111111.00000000 (dalam biner)

-----------------------------------------------------------------

network ID -> 11000000.10101000.00010100.00000000 (dalam biner)

```

Biner 11000000.10101000.00010100.00000000 dapat dikonversi ke decimal untuk mendapatkan network ID-nya. Sehingga, network ID dari IP yang diberikan adalah 192.168.20.0.
