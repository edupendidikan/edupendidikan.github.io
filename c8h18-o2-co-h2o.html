3. cara menyetimbangkan C8H18 +O2 --> CO2+H2O


Nih Jawabannya Random thing here --> lololololololC8H18 + O2 --> CO2 + H20

Pada reaksi di atas :

untuk yg di ruas kiri ( pereaksi / reaktan ) :
Jumlah atom C = 8
H = 18
O = 2

untuk yg di ruas kanan ( produk / hasil ) :
Jumlah atom C = 1
H = 2
O = 3

Untuk menyetimbangkannya maka jumlah masing-masing atom antara ruas kiri dan kanan harus sama ( contohnya Jumlah atom C di ruas kiri sama dengan di ruas kanan, begitu pula untuk unsur H dan O ).

C8H18 + O2 --> CO2 + H20
= C8H18 + O2 --> 8CO2 + H2O ( Diberi koefisien 8 pada CO2 agar jumlah atom C di ruas kiri dan kanan sama yaitu 8 )
= C8H18 + O2 --> 8CO2 + 9H2O ( Diberi koefisien 9 pada H2O agar jumlah atom H pada ruas kiri dan kanan sama yaitu 18 dimana untuk atom H di ruas kanan yakni 9 x 2 : 18 )
= C8H18 + 25/2O2 --> 8CO2 + 9H2O ( karena jumlah atom O di ruas kanan yaitu 8x2 + 9x1 : 25 maka harus di beri koefisien 25/2 pada O2 sebab 25/2 x 2 : 25 )

Namun, perlu diingat bahwa koefisien tidak bisa dalam bentuk pecahan maka harus disetarakan lagi.

C8H18 + 25/2 O2 --> 8CO2 + 9H2O
= 2C8H18 + 25O2 --> 16CO2 + 18H2O ( Masing-masing ruas dikalikan 2 agar pembagi pada 25/2 dapat hilang sebab 25/2 x 2 : 25, sehingga pecahannya sudah tidak ada )

Pada reaksi di atas :

Untuk yg diruas kiri ( reaktan / pereaksi ) :
Jumlah atom C = 16 { 2 x 8 }
H = 36 { 2 x 18 }
O = 50 { 25 x 2 }

Untuk yg diruas kanan ( produk / hasil ) :
Jumlag atom C = 16
H = 36 { 18 x 2 }
O = 50 { ( 16 x 2 ) + 18 }

Karena jumlah atom C, H, dan O di ruas kiri dan kanan sama maka dengan demikian rumus kesetimbangnya ialah :

2C8H18 + 25O2 --> 16CO2 + 18H20

Apa yang dimaksud dengan koefisien reaksi?  

Bilangan atau angka yang berada di depan rumus kimia zat-zat dalam persamaan reaksi disebut dengan koefisien reaksi. Koefisien reaksi ditambahkan untuk menyetarakan atom-atom sebelum dan sesudah reaksi atau sebelum dan sesudah tanda panah. Selain untuk menyetarakan persamaan reaksi, koefisien reaksi menyatakan perbandingan paling sederhana dari partikel zat yang terlibat dalam reaksi.

27. Sebanyak 4,6Liter bensin ( C8H18) dihabiskan oleh sebuah motor untuk menempuh perjalanan sejauh 60 Km perjalanan. Namun karena proses pembakaran tidak berjalan sempurna, maka disamping menghasilkan gas CO2, pembakaran menghasilkan juga gas CO. Berapa gram gas CO yang dihasilkan jika diketahui berat jenis bensin 0,7 Kg/L, bobot atom C = 12 g/mol, H = 1 g/ mol dan O = 16 g/mol Reaksi C8H18 + O2 CO2 + H2O + CO


Untuk menghitung berapa gram gas CO yang dihasilkan, kita perlu menghitung terlebih dahulu berapa mol bensin yang terbakar dan berapa mol gas CO yang dihasilkan.

Langkah-langkahnya adalah sebagai berikut:

Menghitung massa bensin yang terbakar

Massa bensin = volume bensin x berat jenis bensin

Massa bensin = 4,6 L x 0,7 kg/L = 3,22 kg

Menghitung jumlah mol bensin yang terbakar

Mol bensin = massa bensin / massa molar bensin

Mol bensin = 3,22 kg / (8 x 12 g/mol + 18 x 1 g/mol) = 0,141 mol

Menghitung jumlah mol gas CO yang dihasilkan

Berdasarkan reaksi pembakaran di atas, setiap 1 mol bensin akan menghasilkan 1 mol gas CO.

Jumlah mol gas CO = jumlah mol bensin yang terbakar = 0,141 mol

Menghitung massa gas CO yang dihasilkan

Massa gas CO = jumlah mol gas CO x massa molar gas CO

Massa gas CO = 0,141 mol x 28 g/mol = 3,948 g

Jadi, massa gas CO yang dihasilkan oleh pembakaran 4,6 L bensin adalah sebesar 3,948 gram.

30. Persamaan reaksi ; C8H18 + x O2 ------> y CO2 + H2O, jika disetarakan maka nilai x dan y berturut-turut adalah . . . .tolong di jawab:)​


Jawaban:

Maka nilai x dan y adalah 25 dan 16.

Penjelasan:

Untuk penyetaraan reaksi, jumlah kanan harus = jumlah kiri, maka agar menjadi setara:

C8H18 + 25/2 O2 --> 8 CO2 + 9 H2O

Reaksi tersebut koefisiennya bisa dikali 2, agar menjadi bilangan bulat semua. Maka akan menjadi:

2 C8H18 + 25 O2 --> 16 CO2 + 18 H2O

Jumlah C di kiri = 2 × 8 = 16

Jumlah C di kanan = 16 × 1 = 16

Jumlah H di kiri = 2 × 18 = 36

Jumlah H di kanan = 18 × 2 = 36

Jumlah O di kiri = 25 × 2 = 50

Jumlah O di kanan = (16 × 2) + 18 = 32 + 18 = 50

Maka, Jumlah Kanan = Jumlah Kiri

Maka nilai x dan y adalah 25 dan 16.
