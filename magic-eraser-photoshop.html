11. Tuliskan kegunaan ikon berikut ini :1. Ruler tool2. Eyedropper tool3. Healing brush tool4. Brush tool5. Clone stamp tool6. History brush tool7. Eraser tool8. Magic eraser tool9. Gradient tool10. Dodge tool​


Penjelasan:

1. Ruler Tool = berfungsi untuk mengetahi lebar dan tinggi dimensi halaman kerja kita.

2. Fungsi Eyedropper adalah Digunakan untuk mengambil contoh warna pada bagian gambar.

3. Spot healing brush tool

Pada dasarnya Spot healing brush tool berfungsi untuk menghilangkan bagian tertentu pada objek dengan cara menyamakan warna di sekitarnya.

4. Brush tool digunakan untuk membuat polesan kuas. Jika tidak cocok dengan bentuk brush yang telah disediakan Photoshop, kita dapat download bentuk brush sendiri sesuai yang diinginkan. Pencil tool berfungsi untuk membuat objek coretan pensil. Color replacement tool digunakan untuk mengganti gambar pada sebuah gambar.

5. Clone stamp tool berfungsi untuk menggandakan atau /cloning objek atau area yang di inginkan ke posisi lain.

6. Art History Brush Tool berfungsi untuk membuat goresan dengan efek gaya lukisan berdasarkan pada kondisi tertentu pada pallete history, Art History Brush Tool mempunyai option bar yang sama dengan brush tool, dengan tambahan beberapa pilihan.

7. Fungsi Eraser Tool adalah untuk menghapus gambar pada layer di area lembar kerja.

8. Fungsi Magic Eraser Tool adalah untuk menghapus gambar dengan warna solid. dan area gambar yang dihapus menggunakan tool ini akan menjadi transparan.

9. Gradient tool memiliki fungsi untuk membuat warna bergradasi atau lebih dari satu warna dalam satu objek. Tool ini biasanya digunakan untuk memberi warna pada layer atau background gambar.

10. fungsi dodge tool di photoshop adalah menghapus efek shadow pada bagian objek seperti wajah sehingga kita bisa mencerahkan efek gelap tersebut.
